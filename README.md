### Em 2023-2

A empresa Greenneat, especializada em produtos de limpeza sustentáveis, enfrentava um problema na forma de gerenciar usuários e créditos relacionados a transação de troca de óleo. Havia necessidade de acompanhar as transações, a conversão de créditos em desconto na loja virtual e também no controle centralizado para um administrador.  <br>

### Resolução:

Para solucionar o desafio, um aplicação web baseada em React e TypeScript foi desenvolvida. O sistema foi desenvolvido de forma que contas de usuário pudessem ser criadas, transações de óleo usado pudessem ser feitas, créditos gerados e, consequentemente, compras na mesma aplicação web. Houve também a criação de uma página de admnistrador, para que o mesmo pudesse visualizar as compras, movimentação de créditos, entre outros.


[Link do repositório do projeto](https://github.com/TechForce-ADS/Greenneat)

  -  **Node.js**: Configuração do servidor backend.
  -  **MySQL**: Para registro e consulta de dados do sistema.
  -  **Git e Github**: Para versionamento do projeto.
